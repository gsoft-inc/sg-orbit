# Foundation

Once [installed](?path=/docs/introduction-installation--page), Orbit CSS foundation can be customized to your needs. Please be aware that customization would no longuer benefinits from updates and could break.

## Override default variables

The foundation default variables can be overrided by simply redeclaring the variables in a `:root` CSS class:

To override the <A href={getGithubUrl("/packages/foundation/src/core/index.css")} target="_blank" rel="noopener noreferrer">core variables</A>:

```css
:root {
    --primary-50:  #c63a0c;
}
```

To override brand specific variables for <A href={getGithubUrl("/packages/foundation/src/apricot.css")} target="_blank" rel="noopener noreferrer">Apricot</A>, <A href={getGithubUrl("/packages/foundation/src/overcast.css")} target="_blank" rel="noopener noreferrer">Overcast</A> or <A href={getGithubUrl("/packages/foundation/src/desktop.css")} target="_blank" rel="noopener noreferrer">Desktop</A>:

```css
:root {
    --apricot-50: #c63a0c;
    --overcast-50: #c63a0c;
    --desktop-50: #c63a0c;
}
```
