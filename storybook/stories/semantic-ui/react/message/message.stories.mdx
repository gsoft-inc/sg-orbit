# Message

A message displays information that explains nearby content.

## Installation

<Installation imports={<Imports />} />

## Usage

### Default

A default message.

<Preview>
    <Story name="default">
        <Message>
            Easily manage external sharing links and set-up automatic access reviews for entrusted group owners.
        </Message>
    </Story>
</Preview>

### Header

A message can have an header.

<Preview>
    <Story name="header">
        <Message>
            <Message.Header>Setup a security policy for external sharing</Message.Header>
            <p>
                Easily manage external sharing links and set-up automatic access reviews for entrusted group owners.
            </p>
        </Message>
    </Story>
</Preview>

### List

A message can contains a formatted list.

<Preview>
    <Story name="list">
        <Message>
            <Message.Header>New Site Features</Message.Header>
            <Message.List>
                <Message.Item>You can now have cover images on blog pages</Message.Item>
                <Message.Item>Drafts will now auto-save while writing</Message.Item>
            </Message.List>
        </Message>
    </Story>
</Preview>

### Dismissible

A message can be dismissed.

>Dismissible blocks do not automatically close. Use the `onDismiss` callback to remove the message.

<Preview>
    <Story name="dismissible">
        <Message onDismiss={noop}>
            <Message.Header>New Site Features</Message.Header>
            <Message.List>
                <Message.Item>You can now have cover images on blog pages</Message.Item>
                <Message.Item>Drafts will now auto-save while writing</Message.Item>
            </Message.List>
        </Message>
    </Story>
</Preview>

### Info

<Preview>
    <Story name="info">

    </Story>
</Preview>

### Positive / Success

A message can display a positive message.

<Preview>
    <Story name="positive success">
        <Message positive>

### Warning

A message can display a warning message.

<Preview>
    <Story name="warning">

### Negative / Error

A message can display a negative message.

<Preview>
    <Story name="negative error">
        <Message negative>
            <Message.Header>Cannot setup a security policy for external sharing</Message.Header>
            <p>
                Your trial has expired.
            </p>
        </Message>
    </Story>
</Preview>

### Floating

A message can float above content that it is related to.

<Preview>
    <Story name="floating">
        <Message floating>
            <Message.Header>
                Setup a security policy for external sharing
            </Message.Header>
            <p>
                Easily manage external sharing links and set-up automatic access reviews
                for entrusted group owners.
            </p>
        </Message>
    </Story>
</Preview>

### Two Columns Layout

A message can have two columns.
<Preview>
    <Story name="left aligned">
        <Message icon>
            <Button>Click Here</Button>
            <Message.Content>
                <Message.Header>Just one second</Message.Header>
                <p>We are fetching that content for you.</p>
            </Message.Content>
        </Message>
    </Story>
</Preview>

<Preview>
    <Story name="right aligned">
        <Message icon>
            <Message.Content>
                <Message.Header>Just one second</Message.Header>
                <p>We are fetching that content for you.</p>
            </Message.Content>
            <InfoIcon className="w10 h10" />
        </Message>
    </Story>
</Preview>

### Size

A message can vary in size.

Can be set to: `"small"`, or omitted.

<Preview>
    <Story name="size">
        <Message size="small">
            <p className="fw6">
                Easily manage external sharing links and set-up automatic access reviews.
            </p>
        </Message>
    </Story>
</Preview>
