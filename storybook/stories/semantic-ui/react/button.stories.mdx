# Button

## Installation

<details><summary className="outline-0">install the packages</summary>

Install from <a href={INSTALLATION_PAGE}>the React bundle</a> or as a single package:

```bash
npm install @orbit-ui/css-normalize @orbit-ui/tachyons @orbit-ui/semantic-ui-theme semantic-ui-react
```
</details>

<details><summary className="outline-0">then import what you need</summary>

```javascript
    import { Button } from "semantic-ui-react";
```
</details>

## Usage

### Default

A standard button.

<Preview>
    <Story name="default">
        <Button>Cutoff</Button>
    </Story>
</Preview>

#### Active / focus

A button can be focused.

<Preview>
    <Story name="active / focus">
        <Button active>Cutoff</Button>
    </Story>
</Preview>

#### Disabled

A button can show it is currently unable to be interacted with.

<Preview>
    <Story name="disabled">
        <Button disabled>Cutoff</Button>
    </Story>
</Preview>

### Emphasis

A button can be formatted to show different levels of emphasis.

<Preview>
    <Story name="emphasis">
        <div className="flex">
            <Button primary>Primary</Button>
            <Button secondary>Secondary</Button>
        </div>
    </Story>
</Preview>

#### States

A button showing emphasis can have a disabled and an active state.

<Preview>
    <Story name="emphasis states">
        <div className="flex">
            <Button primary disabled>Primary</Button>
            <Button secondary disabled>Secondary</Button>
            <Button primary active>Primary</Button>
            <Button secondary active>Secondary</Button>
        </div>
    </Story>
</Preview>
