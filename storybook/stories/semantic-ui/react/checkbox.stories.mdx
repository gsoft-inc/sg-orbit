# Checkbox

A checkbox allows a user to select a value from a small set of options, often binary.

## Installation

<details><summary >install the packages</summary>

Install from <a href={INSTALLATION_PAGE}>the React bundle</a> or as a single package:

```bash
npm install @orbit-ui/css-normalize @orbit-ui/tachyons @orbit-ui/semantic-ui-theme semantic-ui-react
```
</details>

<details><summary >then import what you need</summary>

```javascript
    import { Checkbox } from "semantic-ui-react";
```
</details>

## Variations

### Toggle

A checkbox can toggle.

<Preview>
    <Story name="toggle">
        <Checkbox toggle label="Milky Way"/>
    </Story>
</Preview>

#### Checked

A toggle can be check by default.

<Preview>
    <Story name="toggle checked">
        <Checkbox toggle defaultChecked label="Milky Way"/>
    </Story>
</Preview>

#### Disabled

A toggle can be disabled.

<Preview>
    <Story name="toggle disabled">
        <Checkbox toggle disabled label="Milky Way"/>
    </Story>
</Preview>

### Radio

A checkbox can be formatted as a radio element. This means it is an exclusive option.

<Preview>
    <Story name="radio">
        <Checkbox radio label="Milky Way"/>
    </Story>
</Preview>

#### Disabled

A radio element can be disabled.

<Preview>
    <Story name="radio disabled">
        <Checkbox radio disabled label="Milky Way"/>
    </Story>
</Preview>

### Radio Group

Radios in a group must be controlled components.

<Preview>
    <Story name="radio group">
        <ControlledRadioGroup />
    </Story>
</Preview>

### Fitted

A fitted Checkbox does not leave padding for a label.
