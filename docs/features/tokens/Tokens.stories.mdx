# Tokens

Tokens are Orbit's reusable values, providing a common language between developers and designers.

Aliased tokens are named for their use case, rather than their value. For example `alias-alert` is named to indicate the appropriate background color for an error message background, as opposed to `red-5` which is less specific and could be used in a number of ways depending on context.

This makes their intended use clear and intentional, and allows us to develop a scalable and consistent visual language while also being the building block of Orbit's [theming](/docs/theming--page).

In Orbit, **1 rem unit = 16 pixels**.

## Usage

Tokens can be used as `CSS` variables or [Style Props](/docs/style-props--page) values.

Style Props:

```jsx
    <Div backgroundColor="alias-accent-faint">...</Div>
```

CSS Variable:

```css
    .item {
        background-color: var(--o-ui-bg-alias-accent-faint);
    }
```
