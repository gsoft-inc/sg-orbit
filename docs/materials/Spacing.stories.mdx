# Spacing

Orbit Design spacing scale use an 8 point scale.

We chose a base-8 scale because eight is a highly composable number (it can be divided and multiplied many times and result in whole numbers), yet allows spacing dense enough for ShareGate's products.

A few half points have been added to cover specific use cases that requires smaller padding and margin.

### Units

Our spacing values are expressed in `rem` units based on a font-size of `16px`. We choosed `rem` units over `em` units since they are relative to the font-size of the root element instead of the parent element which can cause unexpected results when cascading.

For a deep understanding of rem units, refer to the [Values and units](https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Values_and_units) article on MDN.

## Values
