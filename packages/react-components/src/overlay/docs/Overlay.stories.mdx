## Usage

### Example

An overlay takes care of rendering a container for your component with Orbit's *enter* & *leave* default animations and enables theme support for the overlay content.

The visibility of the overlay can be toggled by providing a `show` prop.

```jsx
<Overlay show>
    SpaceX designs, manufactures, and launches the worldâ€™s most advanced rockets and spacecraft.
    The company was founded in 2002 by Elon Musk to revolutionize space transportation, with the ultimate goal of making life multiplanetary.
</Overlay>
```
