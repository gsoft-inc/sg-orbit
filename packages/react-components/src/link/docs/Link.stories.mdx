# Link

<Tagline>Links allow a user to navigate between pages, scroll to a have page section, access files, or even to handle email addresses.</Tagline>

<ComponentInfo
    usage={"import { TextLink, IconLink, Link } from \"@orbit-ui/react-components\""}

## Usage

### Default

A default link.

<Preview>
    <Story name="default">
        <TextLink href="#">Flight details</TextLink>
    </Story>
</Preview>

### Icon

A link can contain text and [icons](#).

<Preview>
    <Story name="icon">
        <TextLink href="#">
            <Text>Flight details</Text>
            <WarningIcon />
        </TextLink>
    </Story>
</Preview>

### Left icon

*Non standard* left [icons](#) can be provided to handle special cases. However, think twice before adding *left icons*, *right icons* should be your go to.

<Preview>
    <Story name="left icon">
        <TextLink href="#">
            <WarningIcon slot="left-icon" />
            <Text>Flight details</Text>
        </TextLink>
    </Story>
</Preview>

### External

A link can indicate that his destination is an external resources.

An external link will render with `target="_blank"` and `rel="noopener noreferrer"` attributes. These attributes values can be overrided by specifying a `target` or `rel` prop.

<Preview>
    <Story name="external">
        <TextLink external href="#">Flight details</TextLink>
    </Story>
</Preview>

### Color

A link can use a different color accent.

<Preview>
    <Story name="color">
        <Inline>
            <TextLink color="primary" href="#">Primary</TextLink>
            <TextLink color="secondary" href="#">Secondary</TextLink>
            <TextLink color="danger" href="#">Danger</TextLink>
        </Inline>
    </Story>
</Preview>

### Disabled

A link can have *disabled* style, it's your responsability to make sure the link doesn't navigate anywhere when activated.

<Preview>
    <Story name="disabled">
        <TextLink disabled href="#">Flight details</TextLink>
    </Story>
</Preview>

### Size

A link can vary in size.

<Preview>
    <Story name="size">
        <Inline verticalAlign="center">
            <TextLink size="sm" href="#">Flight details</TextLink>
            <TextLink href="#">Flight details</TextLink>
        </Inline>
    </Story>
</Preview>

### Button

A link can behave like a button.

<Preview>
    <Story name="button">
        <TextLink as="button">Flight details</TextLink>
    </Story>
</Preview>

### React router

To integrate with a third-party routing library like [React Router](https://reactrouter.com) any link component can be rendered as a [React Router Link](https://reactrouter.com/web/api/Link) or as any other custom component.

<Preview scope={{ RouterLink }}>
    <Story name="react router">
        <TextLink as={RouterLink} to="/flight-details">Flight details</TextLink>
    </Story>
</Preview>

## Icon link

A link content can be a single icon.

> When using this variant, an accessible name must be provided through `aria-label` prop. See [WCAG practices](https://www.w3.org/TR/WCAG20-TECHS/ARIA6.html).

<Preview>
    <Story name="icon link">
        <Inline verticalAlign="center">
            <IconLink size="sm" aria-label="Add" href="#"><AddIcon /></IconLink>
            <IconLink aria-label="Add" href="#"><AddIcon /></IconLink>
        </Inline>
    </Story>
</Preview>

## Basic link

A link can have a more subtle appareance and only be noticeable when focused.

### Image

A link content can be an image element.

<Preview scope={{ LaunchImage }}>
    <Story name="basic link image">
        <Link href="#">
            <Image src={LaunchImage} alt="SpaceX launch" width="325px" height="216px" />
        </Link>
    </Story>
</Preview>

### Shape

A link can use a different shape to match his content shape on focus state.

<Preview scope={{ LaunchImage }}>
    <Story name="basic link shape">
        <Inline>
            <Link shape="rounded" focus href="#">
                <Image src={LaunchImage} alt="SpaceX launch" width="100px" height="100px" />
            </Link>
            <Link shape="circular" focus href="#">
                <Image src={LaunchImage} alt="SpaceX launch" width="100px" height="100px" />
            </Link>
            <Link shape="box" focus href="#">
                <Image src={LaunchImage} alt="SpaceX launch" width="100px" height="100px" />
            </Link>
        </Inline>
    </Story>
</Preview>
